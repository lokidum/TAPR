# TAPR — Feature Backlog

Features not yet built, ordered by priority/phase where applicable.

---

## Phase 3 and Beyond

### AR Style Preview

Augmented reality try-on for haircuts and styles. Allow consumers to preview how a cut would look before booking.

### Supply Chain Wholesale Ordering

Bulk ordering for studios and barbers — products, equipment, consumables. Integration with suppliers and inventory management.

### Full TikTok/Instagram API Auto-Sync

Automatic sync of barber portfolio content from TikTok and Instagram. Currently supports manual linking; full API integration would enable auto-import of posts and reels.

### Interstate Guest Spotlight

Feature barbers who travel or work across state borders. Highlight "guest" or "visiting" barbers for studios and consumers.

### Banuba Video Editor SDK Integration

In-app video editing for portfolio content. Trim, filters, and basic edits before upload.

### ACT Portable Long Service Leave Compliance

Australian Capital Territory portable long service leave compliance for the hairdressing and barbering industry. Tracking, reporting, and levy management.

### Admin Web Panel (Next.js)

Full admin dashboard for:
- User management (ban/unban, roles)
- Dispute resolution
- Content moderation
- Analytics and reporting
- System configuration

See `apps/admin` (not started).

### Scalp Analysis Tools

AI or expert-assisted scalp analysis. Photos, condition assessment, product recommendations.

### AI-Powered Maintenance Intelligence

Predictive maintenance for equipment, inventory replenishment, and appointment no-show risk. Recommendations based on historical data.
